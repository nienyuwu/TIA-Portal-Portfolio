# TIA Portal Portfolio

A collection of industrial automation projects developed with **Siemens TIA Portal V16**, **PLCSIM**, and **Factory I/O**.

The purpose of this repository is to document my PLC programming journey and showcase practical automation projects, from basic control logic to more advanced industrial applications.

---

## Technologies

- Siemens TIA Portal V16
- SIMATIC S7-1200 PLC
- PLCSIM
- Factory I/O
- Ladder Logic (LAD)

---

## Projects

| Project | Description | Status |
|---------|-------------|:------:|
| **Project 01 - From A to B** | Conveyor object transfer using sensors and actuators | ✅ Completed |
| **Project 02 - Tank Filling** | Automatic tank filling system with level monitoring | ✅ Completed |

More projects will be added over time.
