## Overview

Project 02 simulates an automatic tank filling process using Siemens TIA Portal V16 and Factory I/O.

The objective is to build a complete automatic filling sequence, including bottle detection, conveyor control, filling control, timers, and process interlocks.

This project is part of my Siemens PLC Portfolio.

---

## Development Environment

- Siemens TIA Portal V16 Professional
- Siemens S7-PLCSIM
- Factory I/O
- Siemens S7-1200 CPU

---

## Project Goals

- Automatic conveyor control
- Bottle detection
- Tank filling sequence
- Level sensor control
- Automatic cycle operation
- Alarm and reset functions
