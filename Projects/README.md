# Project 01 - From A to B

## Overview

This project demonstrates a basic conveyor control system developed using Siemens TIA Portal V16, S7-PLCSIM, and Factory I/O.

The conveyor transports a box from point A to point B. When the photo sensor detects the box, the conveyor stops.

---

## Objectives

- Learn Factory I/O communication
- Learn Siemens S7-1200 programming
- Learn basic Ladder Logic
- Build a reusable PLC project architecture
